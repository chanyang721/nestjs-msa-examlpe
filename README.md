## Directory Structure
    - document: 서비스 관련 문서화 파일
    - apps: 서비스 도메인 모듈
    - libs: 서비스 실행 시 필요한 라이브러리 모듈
    - packages: 서비스 실행 시 필요한 package.json 모음
    - test
        - domain: 서비스 도메인 모듈 테스트 코드
            - e2e: e2e 테스트 코드
            - unit: unit 테스트 코드
        - lib: 서비스 실행 시 필요한 라이브러리 모듈 테스트 코드
        - utils: 테스트 시 필요한 메서드 모음
